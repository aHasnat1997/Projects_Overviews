Case Study

MyStudyPal

The personal AI study buddy for Irish students. Providing instant explanations, exam-style answers, structured notes, smart quizzes, and progress insights, fully aligned with the Irish curriculum.

Role: Deployment & Automation Engineer Scope: AI Educational Platform Focus: DevOps + Automation Scripts
Highlights

7 distinct microservices managed entirely via Dokploy, custom Puppeteer scraping pipeline for automated past paper ingestion, and high-performance Rustfs media storage.

Docker Dokploy React NestJS Puppeteer Rust PostgreSQL Redis

Overview

MyStudyPal accelerates learning by providing a highly contextual environment where students can upload complex problems and receive step-by-step logic. Rather than giving generic AI answers, the platform guarantees grading-scheme-aligned feedback for subjects like Maths, Science, English, and Irish.

Vision

Replace unstructured study methods with a targeted, curriculum-specific AI tool that helps students generate concise notes, test their knowledge, and actively identify weak points.

Technical Innovation

A robust 7-container microservice infrastructure orchestrated via Dokploy, coupled with a custom Puppeteer script that autonomously scrapes official past paper data to continuously tune the AI.

Outcome

A stable, production-deployed platform with automated data ingestion pipelines, high-speed file serving via Rustfs, and a centralized monitoring dashboard ensuring high availability.

Tech Stack & Infrastructure

A highly decoupled microservices architecture designed for isolation, security, and independent scalability.

Frontend & Dashboards React (User Dashboard, Admin Dashboard), HTML (Landing Page)
Backend & API NestJS (Core Server API), Python (AI Processing)
Databases & Storage Postgres-DB (Relational Data), Redis-db (Caching & Sessions), Rustfs (High-speed Rust media server for image uploads)
Automation & Scraping Puppeteer, Node.js (Autonomous past paper ingestion pipeline)
Deployments & DevOps Docker, Dokploy Container Orchestration, Nginx/Traefik

Key Features

  • AI Study Buddy: Ask anything across Maths, Science, English, and Irish. Get clear, exam-style answers with step-by-step logic aligned with marking schemes. Supports image uploads.
  • Notes Generator: Pick a subject or topic and instantly generate clean, structured notes based entirely on the Irish curriculum. Perfect for revision.
  • Smart Quizzes: AI dynamically builds quizzes (multiple choice or written) around a student's weak areas to help them improve faster, complete with weak topic tagging.
  • Progress & Insights: A clean dashboard tracking improvements, featuring an XP level system, weekly improvement stats, and a strength/weakness heatmap.
  • Automated Data Pipeline: Custom scrapers ensure the AI backend is continuously fed with the latest past papers and marking schemes.

My Role & Contributions

  • Designed and managed the entire microservices deployment strategy, strictly utilizing Docker to containerize the React dashboards, static HTML, NestJS backend, and Rust file server.
  • Orchestrated the complete production ecosystem using Dokploy, configuring isolated networks, persistent volumes for Postgres and Redis, and real-time container health monitoring.
  • Built a robust web automation script using Puppeteer that crawls the official Irish curriculum platforms to scrape past paper questions, images, and grading rubrics.
  • Piped the scraped structured data directly into the NestJS backend to ensure the AI maintained absolute curriculum accuracy without relying on manual data entry.
  • Managed environment variables, secrets, and automated build pipelines to guarantee zero-downtime updates across all 7 containers.
